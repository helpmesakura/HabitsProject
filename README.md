# Разработка сервиса для борьбы с вредными привычками "Tix"
## 1. Концепция проекта
Название: ***Tix***

Основная цель: Проектирование и разработка сервиса для помощи пользователям в борьбе с вредными привычками.

Ключевая идея: Сервис фокусируется на решении проблем зависимостей и заинтересованности в них, помогая человеку справиться с вредной привычкой за счет системы целей, отслеживания прогресса и мотивации.

Целевая аудитория: Пользователи в возрасте 14-75 лет.

Уникальное торговое предложение (УТП): На рынке отсутствуют прямые аналоги. Существующие решения являются либо платными, либо имеют урезанный функционал в бесплатной версии, так как их бизнес-модель ориентирована на прибыль. Наш продукт полностью бесплатен.

## Цель и задачи проекта
Цель: Проектирование и разработка сервиса для помощи с вредными привычками.

Задачи:

Проанализировать предметную область.

Определить требования и сформировать ТЗ.

Спроектировать сервис.

Разработать MVP с внешними интеграциями.

Провести тестирование (ручное и автоматизированное).

Написать пользовательскую документацию.

## Функциональные и нефункциональные требования
### Функциональные требования:

Управление привычками (создание, переименование, удаление).

Управление целями.

Система наград за выполнение целей.

Генерация ссылок для обмена прогрессом.

Поиск привычек по названиям.

Регистрация и авторизация пользователей.
### Нефункциональные требования:
Минималистичный и интуитивно понятный дизайн.

Адаптивная верстка (для мобильных и десктопных устройств).

Обязательное шифрование паролей пользователей.

Использование облачного хранилища для данных.

Возможность использования основного функционала без регистрации.
